# 🌱 AI Smart Agriculture App

## 📌 Overview
This project is an AI-based Smart Agriculture Application that helps farmers and users choose the best and most profitable crop based on soil and environmental conditions.

The system uses Machine Learning along with real-time weather data to provide intelligent crop recommendations.

---

## 🚀 Features

- 🌾 Crop Prediction using Machine Learning
- 🌦️ Real-time Weather Data (Temperature & Humidity)
- 💰 Profit Estimation for Crops
- 📊 Graph Visualization of Top Crop Predictions
- 🗺️ Map-based Location Display
- 🤖 AI Chatbot for Farming Assistance
- 📱 Mobile-friendly Web Application

---

## 🧠 Technologies Used

- Python
- Streamlit (Web App Framework)
- Scikit-learn (Machine Learning)
- Pandas & NumPy (Data Processing)
- OpenWeatherMap API (Real-time Data)

---

## 📊 Input Parameters

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH Value
- Rainfall

---

## 📈 Output

- Recommended Crop 🌾
- Estimated Profit 💰
- Top 3 Crop Suggestions 📊
